# SPANISH
# Veterinaria José Granda - React

Sitio web oficial de la Veterinaria José Granda construido con React.

## Características
- Diseño completamente responsivo para todo dispositivo.
- Animaciones al hacer scroll para una experiencia dinámica de usuario.
- Menú hamburguesa funcional para navegación móvil.
- Integración con Google Maps para localizar fácilmente la clínica.
- Enlaces a redes sociales para comunicación directa

## Instalación

1. Clonar el repositorio
2. Instalar dependencias:
bash
    npm install
3. Correr el proyecto:
bash
    npm run dev
